Q4. Regular Grammar

Definition

A Regular Grammar is a formal grammar (G = (V, Σ, P, S)) that generates regular languages. It is a Type-3 grammar in the Chomsky hierarchy and can be processed by finite automata.

Four Types of Regular Grammar

1. Right Linear Grammar (Right Regular Grammar)

Production Rules:

  • A → aB (non-terminal followed by terminal and non-terminal)
  • A → a (non-terminal to terminal)
  • A → ε (non-terminal to empty string)
Example:
S → aS | bA | ε
A → aA | b

This generates strings like: ε, b, ab, aab, aaab, bb, abb, etc.

2. Left Linear Grammar (Left Regular Grammar)

Production Rules:

  • A → Ba (non-terminal and terminal following non-terminal)
  • A → a (non-terminal to terminal)
  • A → ε (non-terminal to empty string)
Example:
S → Sa | Ab | ε
A → Aa | b

This generates strings like: ε, b, ba, baa, baaa, bb, bba, etc.

3. Strictly Right Linear Grammar

Production Rules:

  • A → aB (must have non-terminal on right)
  • S → ε (only start symbol can produce ε)
Example:
S → aA | bB | ε
A → aA | bB
B → aA | bB

More restrictive than right linear grammar.

4. Strictly Left Linear Grammar

Production Rules:

  • A → Ba (must have non-terminal on left)
  • S → ε (only start symbol can produce ε)
Example:
S → Aa | Bb | ε
A → Aa | Bb
B → Aa | Bb

More restrictive than left linear grammar.

Key Properties

Property Description
Language Class Generates Regular Languages only
Automaton Equivalent Finite Automata (DFA/NFA)
Chomsky Hierarchy Type-3 Grammar
Closure Properties Closed under union, concatenation, Kleene star
Limitations Cannot handle nested structures or counting

Q5. Arden's Theorem Proof

Theorem Statement

Let P and Q be two regular expressions over alphabet Σ. If P does not contain ε (empty string), then the equation:

R = Q + RP

has a unique solution:

R = QP*

where P* = ε + P + PP + PPP + ... (zero or more concatenations of P)

Proof

Part 1: Show R = QP* is a solution

Substitute R = QP* into R = Q + RP: LHS = R = QP* RHS = Q + RP = Q + (QP*)P = Q + Q(P*P) = Q + QP⁺ [where P⁺ = PP*] = Q(ε + P⁺) = Q(ε + P + PP + PPP + ...) = QP* Therefore: LHS = RHS Hence, R = QP* satisfies the equation. ✓

Part 2: Prove Uniqueness

Assume there exists another solution R' such that R' = Q + R'P

Repeatedly substitute R' on the right side: Iteration 0: R' = Q + R'P Iteration 1: R' = Q + (Q + R'P)P = Q + QP + R'P² Iteration 2: R' = Q + QP + (Q + R'P)P² = Q + QP + QP² + R'P³ Iteration n: R' = Q + QP + QP² + ... + QP^n + R'P^(n+1) = Q(ε + P + P² + ... + P^n) + R'P^(n+1) As n → ∞: Since P does not contain ε, |P| ≥ 1 Therefore, P^(n+1) generates strings of length ≥ n+1 As n → ∞, R'P^(n+1) → ∅ (vanishes) Therefore: R' = Q(ε + P + P² + P³ + ...) = QP*

Conclusion

Since any solution R' must equal QP*, the solution is UNIQUE.

Hence Proved: R = Q + RP has unique solution R = QP* when P doesn't contain ε

Important Condition

⚠️ Critical Condition: P must NOT contain ε (empty string)

Why? If P contains ε, then:
• P* = ε, making the solution trivial
• The equation may not converge
• Multiple solutions may exist
• The iterative substitution method fails

Application Example

Example: Solve R = a + Rb

Here: Q = a, P = b (P doesn't contain ε ✓)

Using Arden's Theorem:
R = QP* = a(b)* = ab*

This means R accepts: a, ab, abb, abbb, ...
